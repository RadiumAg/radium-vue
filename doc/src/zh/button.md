#### Button Api

| 参数       | 说明           | 类型    | 可选值                                   | 默认值 |
| :--------- | :------------- | ------- | :--------------------------------------- | ------ |
| size    | 按钮尺寸       | string  | medium/small/mini                        | ——     |
| type    | 按钮类型       | string  | primary/success/warning/danger/info/text | ——     |
| plain   | 是否为朴素按钮 | boolean | ——                                       | ——     |
| round   | 是否为圆角     | boolean | ——                                       | false  |
| circle  | 是否为圆形按钮 | boolean | ——                                       | false  |
| loading | 是否加载中状态 | boolean | ——                                       | false  |
| icon    | 图标类名       | string  | ——                                       | ——     |
